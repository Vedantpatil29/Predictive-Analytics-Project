# Predictive Analytics Using Historical Data

## Objective
To predict future sales using historical sales data with the help of Machine Learning.

## Tools Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib

## Dataset
The dataset contains monthly sales data.

| Month | Sales |
|---------|---------|
| 1 | 120 |
| 2 | 135 |
| 3 | 150 |
| ... | ... |
| 12 | 290 |

## Steps Performed

### Step 1: Import Libraries
Imported pandas, matplotlib, and Linear Regression model.

### Step 2: Load Dataset
Loaded the sales_data.csv file using pandas.

### Step 3: Data Cleaning
Checked for missing values and cleaned the dataset.

### Step 4: Model Training
Trained a Linear Regression model using historical sales data.

### Step 5: Prediction
Predicted sales for Month 13.

### Step 6: Visualization
Created a graph showing the relationship between Month and Sales.

## Output
Predicted Sales for Month 13: 304.54

## Conclusion
The model successfully learned from historical sales data and predicted future sales trends.
